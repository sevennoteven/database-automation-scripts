# Database Automation Scripts

Collection of PowerShell automation scripts for database administration across MySQL, SQL Server, and PostgreSQL.

## Scripts

### Database Health Check (In Progress)
Multi-platform database health monitoring script.

**Features:**
- Connection testing
- Disk space monitoring
- Database size reporting
- Long-running query detection

**Supports:**
- MySQL
- SQL Server
- PostgreSQL

## Project Structure
```
/scripts - PowerShell automation scripts
/docs - Documentation and guides
/examples - Example configurations and outputs
```
